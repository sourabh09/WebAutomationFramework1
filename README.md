## Web Automation Framework

Automation suite for web automation

## Tech stack
* Written in JAVA
* Implemented using TestNG
* Build Tool - Maven
* Implemented Page Object Model Design Pattern using PageFactory
* Extent Reporting 
* Log4j2 Implemented for logging

## Browsers Supported

* Google Chrome
* Microsoft Edge

## Report details
